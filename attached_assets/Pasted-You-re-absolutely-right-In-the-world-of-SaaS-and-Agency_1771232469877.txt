* **The "God" Move:** If they type something you *don't* have (e.g., "Mobile Axe Throwing Company" or "Luxury Exotic Car Rental"), the system says: *"I don't have a template for that yet—generating your custom blueprint now."*

### 2. The Code: The "Blueprint Generator"

This script takes any niche and builds the entire sub-account structure (Pipelines, SMS, Voice Agent) in about 3 seconds.

```javascript
// lib/ai-blueprint-generator.js
import OpenAI from 'openai';

export async function generateInfiniteNiche(userNicheInput) {
  const prompt = `
    Act as a Master Business Architect. 
    Create a complete business blueprint for a "${userNicheInput}" company.
    Return a JSON object with:
    1. "pipeline": 5 stages (e.g., Lead, Quote Sent, Booked, Completed, Review Requested)
    2. "voice_agent": A 3-paragraph persona for a Vapi agent.
    3. "workflows": 3 automation triggers (Missed Call, New Lead, Review Request)
    4. "suggested_vibe": Which of our themes fits best? (cyber-glass, neon-nights, clean-glass)
  `;

  const completion = await openai.chat.completions.create({
    model: "gpt-4-turbo",
    messages: [{ role: "system", content: prompt }],
    response_format: { type: "json_object" }
  });

  return JSON.parse(completion.choices[0].message.content);
}

```
