### 1. What is the "Liquid Website" Logic?

In a standard site, the colors are hard-coded. In a **Liquid Website**, the entire UI is controlled by **URL Parameters** or **Database States**.

* **The Feature:** You can append `?vibe=neon-nights` to any URL, and the site instantly shifts from a clean medical look to a dark, high-contrast creator look.
* **The "God Mode" Use Case:** An agency owner can show a client 5 different "vibes" for their business in 5 seconds just by switching a toggle, without ever touching the code.

### 2. The Code: The "Vibe Engine" (CSS Variables)

To make this scale to infinite niches, you don't write 100 CSS files. You write **one** system that uses "CSS Variables" (Custom Properties).

```javascript
// lib/vibe-engine.ts

const VIBE_PALETTES = {
  "neon-nights": {
    "--primary-glow": "#00f3ff",   // Cyber Blue
    "--bg-base": "#050505",       // Deep Void Black
    "--accent-color": "#ff007a",  // Neon Pink
    "--glass-blur": "20px",
    "--font-style": "'Orbitron', sans-serif"
  },
  "gilded-rose": {
    "--primary-glow": "#ffd700",   // Gold
    "--bg-base": "#0a0505",       // Deep Wine Black
    "--accent-color": "#ff3366",  // Rose
    "--glass-blur": "12px",
    "--font-style": "'Playfair Display', serif"
  },
  "clean-glass": {
    "--primary-glow": "#ffffff",   // Pure White
    "--bg-base": "#f0f2f5",       // Light Frost (Light Mode)
    "--accent-color": "#007aff",  // Apple Blue
    "--glass-blur": "40px",
    "--font-style": "'Inter', sans-serif"
  }
};

export function applyVibe(vibeName) {
  const palette = VIBE_PALETTES[vibeName] || VIBE_PALETTES["neon-nights"];
  
  // This "pours" the new colors into the site instantly
  Object.entries(palette).forEach(([property, value]) => {
    document.documentElement.style.setProperty(property, value);
  });
}

```
