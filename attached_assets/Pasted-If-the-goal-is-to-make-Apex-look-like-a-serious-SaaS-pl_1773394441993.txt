# 1. WhatsApp Business API

This is massive globally.

Why it matters:

* Open rates **80–95%**
* Huge for **real estate, home services, international leads**
* Businesses trust WhatsApp more than SMS

Example Apex automation:

```
Facebook Lead Ad
→ Apex CRM
→ WhatsApp message
→ AI response
→ Calendar booking
```

Twilio actually supports WhatsApp already, so this is a **fast integration**.
