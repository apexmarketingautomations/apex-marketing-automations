WHAT TO BUILD
Build 3 monetization features into the checkout funnel:

1. ORDER BUMP ON CHECKOUT

* Add a pre-purchase order bump directly on the checkout page
* Offer name: Premium Upgrade
* Price: $9.99 one-time
* Position: directly above the final payment button
* Style: checkbox add-on with a short benefit list
* Copy:

  * “Upgrade Your Card for Maximum Impact”
  * “Enhanced design customization”
  * “Priority support”
  * “Optimized layout for better conversions”
  * “Add Premium for just $9.99”
* Behavior:

  * unchecked by default
  * when checked, total updates instantly
  * selected bump must be included in the final order metadata
  * must persist through payment and confirmation

2. POST-PURCHASE ONE-CLICK UPSELL

* After successful purchase of the base card, show a post-purchase upsell page before the final thank-you page
* Offer name: Pro Business Bundle
* Price: $19.99 one-time
* Copy:

  * “Everything in Premium plus branding help and setup guidance”
  * “Custom branding help”
  * “Setup guidance for your business”
  * “Optimization tips for more leads”
* CTA buttons:

  * primary: “Add to My Order”
  * secondary: “No Thanks”
* Behavior:

  * if accepted, append to the original order
  * if declined, continue to thank-you page
  * store acceptance/decline event for analytics

3. REFERRAL TRACKING

* Keep and extend referral tracking using URL params like ?ref=CODE
* On landing:

  * capture referral code
  * store it in session/local storage
  * carry it through checkout and purchase
* On completed purchase:

  * attach referral code to order record
  * make it available for future commission logic
* Do not fake commission payouts or stats if backend logic does not exist yet
* If full payout automation is not built, clearly structure the data model so it can be added later
