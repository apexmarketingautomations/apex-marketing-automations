### **1. The Master Profit Report Logic**

This script aggregates the data from your global ledger and calculates your "Net Gain" from markups and ad clicks.

```javascript
// Add to your server.js or a separate /api/admin/reports route
app.get('/api/admin/profit-report', ensureRootAccess, async (req, res) => {
    const totalMarkupProfit = await db.get('global_platform_profit') || 0;
    const adRevenue = await db.get('global_ad_revenue') || 0;
    
    // Calculate weekly trends (assuming you log timestamps for each profit entry)
    const weeklyData = await getWeeklyProfitStats(); 

    res.json({
        totalRevenue: totalMarkupProfit + adRevenue,
        markupProfit: totalMarkupProfit,
        internalAdProfit: adRevenue,
        activePaidWallets: await countActivePaidUsers(),
        weeklyTrend: weeklyData
    });
});

```
