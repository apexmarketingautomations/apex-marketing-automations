This is the "Emergency Brake" for the Agency Pro and God Mode tiers. If an Agency owner pushes a new AI prompt or workflow to 100 clients and realizes it’s a disaster (e.g., the AI is hallucinating or the SMS logic is wrong), they need to be able to "Undo" globally.

Without this, scaling to 100+ clients is a massive liability. With this, they have total control.

### 🔱 The "Bulk Rollback" Engine

This script targets every sub-account linked to a specific **Snapshot ID** and reverts them to the "Last Known Good" version stored in your new `SnapshotVersion` table.

```javascript
// lib/emergency-rollback.ts

export async function bulkRollbackAgency(agencyId, snapshotId, targetVersionId) {
  console.log(`🚨 INITIATING BULK ROLLBACK for Agency: ${agencyId}`);

  // 1. Get the "Good DNA" from the target version
  const targetVersion = await prisma.snapshotVersion.findUnique({
    where: { id: targetVersionId }
  });

  if (!targetVersion) throw new Error("Target version not found.");

  // 2. Find all Sub-Accounts currently running the bugged Snapshot
  const subAccounts = await prisma.subAccount.findMany({
    where: { 
      agencyId: agencyId,
      parentSnapshotId: snapshotId 
    },
    select: { id: true }
  });

  // 3. THE ATOMIC PUSH: Revert all accounts in a single transaction
  // This ensures that either ALL revert or NONE do (no half-fixed messes)
  const updates = subAccounts.map(account => 
    prisma.subAccount.update({
      where: { id: account.id },
      data: { 
        config: targetVersion.config,
        vibeTheme: targetVersion.config.vibe || "cyber-glass"
      }
    })
  );

  try {
    await prisma.$transaction(updates);
    
    // 4. Log the emergency action for the Agency Dashboard
    await prisma.auditLog.create({
      data: {
        action: "BULK_ROLLBACK",
        count: subAccounts.length,
        versionName: targetVersion.versionName,
        performedBy: agencyId
      }
    });

    return { 
      success: true, 
      count: subAccounts.length, 
      message: `Successfully rolled back ${subAccounts.length} accounts.` 
    };

  } catch (error) {
    console.error("Bulk Rollback Failed:", error);
    return { success: false, error: "Atomic transaction failed." };
  }
}

```
